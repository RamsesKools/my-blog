# Obsidian

[Obsidian](https://obsidian.md) is a note-taking app built around local Markdown files.

<!-- more -->

No proprietary format, no lock-in, no cloud dependency.
Your notes are just files on disk — readable and editable with any text editor, now and in ten years.

What sets it apart from other note-taking tools is how it treats linking.
Creating a link between notes is frictionless, and the graph view makes it easy to see how ideas connect over time.
The plugin ecosystem is vast, and the core app stays fast and minimal.

The philosophy behind it — local-first, plain files, built to last — is a direct reflection of how its CEO [Steph Ango](steph-ango.md) thinks about software.
